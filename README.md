Unity Extensions
===

### Install via git URL

Requires a version of unity that supports path query parameter for git packages (Unity >= 2019.3.4f1, Unity >= 2020.1a21). You can add `https://github.com/DanilLamov/UnityExtensions/tree/master/Assets/Lamov/UnityExtensions` to Package Manager
